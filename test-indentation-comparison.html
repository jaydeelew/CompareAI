Python Mergesort - Correct vs Incorrect Indentation

This page demonstrates the difference between correct and incorrect Python indentation in mergesort implementations.

❌ INCORRECT Indentation (What the AI model generates)

def mergesort_inplace(arr):
    """In-place mergesort implementation."""
    if len(arr) <= 1:
        return arr
    
    mergesort_helper(arr, 0, len(arr) - 1)  # ← WRONG: Should be at base level
    return arr                               # ← WRONG: Should be at base level

def mergesort_helper(arr, left, right):
    if left < right:
        mid = (left + right) // 2
        mergesort_helper(arr, left, mid)
        mergesort_helper(arr, mid + 1, right)
        merge_inplace(arr, left, mid, right)

def merge_inplace(arr, left, mid, right):
    # Create temporary arrays
    left_arr = arr[left:mid + 1]
    right_arr = arr[mid + 1:right + 1]
    i = j = 0
    k = left
    
    while i < len(left_arr) and j < len(right_arr):
        if left_arr[i] <= right_arr[j]:
            arr[k] = left_arr[i]
            i += 1
        else:
            arr[k] = right_arr[j]
            j += 1
        k += 1
    
    while i < len(left_arr):                # ← WRONG: Should be at same level as first while
        arr[k] = left_arr[i]               # ← WRONG: Should be at same level as first while
        i += 1                             # ← WRONG: Should be at same level as first while
        k += 1                             # ← WRONG: Should be at same level as first while
    
    while j < len(right_arr):              # ← WRONG: Should be at same level as first while
        arr[k] = right_arr[j]              # ← WRONG: Should be at same level as first while
        j += 1                             # ← WRONG: Should be at same level as first while
        k += 1                             # ← WRONG: Should be at same level as first while

Key Differences

  • Function body indentation: After the if block ends, subsequent statements should be at the base level (no indentation)
  • Loop continuation: Multiple while loops at the same level should have the same indentation
  • Block structure: Python uses indentation to define code blocks - incorrect indentation breaks the logical structure
